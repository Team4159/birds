Programming with Brandon Part 2


Let's start with the basics. Learning all the programming languages you can won't make you a better programmer. 90% of all programming languages are pretty much the same logically. The only thing that distinguishes programming languages are their usages and syntax.

Soo... let's say that you know the logic. If you haven't yet, learn variables, constants, arrays, strings, expressions, statements, functions, program flow, pointers, objects, inheritance, polymorphism... yea whatever etc. Great! Now let's learn what's different from all programming languages: their syntax.

The preferred way of learning code you don't know is by consulting Stack Overflow. As said by Tariz Ali, 'Stack Overflow is an essential tool for the modern programmer. Learn how to utilize the latest in "crowd sourcing" technologies to help meet management deadlines.' Amazing, huh?
